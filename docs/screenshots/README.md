## Screenshot Guidelines

1. **Resolution**: Use a standard desktop resolution (1920x1080 or similar)
2. **Browser**: Chrome or Firefox recommended
3. **Language**: Set to English for Riot review
4. **Data**: Use real data (not empty states) when possible
5. **Format**: PNG format preferred
6. **Size**: Ensure text is readable (not too small)

## How to Take Screenshots

### macOS
- Full screen: `Cmd + Shift + 3`
- Selected area: `Cmd + Shift + 4`
- Window: `Cmd + Shift + 4`, then `Space`, then click window

### Windows
- Full screen: `Windows + PrtScn`
- Selected area: `Windows + Shift + S`

### Browser DevTools
1. Open DevTools (F12)
2. Click device toolbar icon
3. Set to "Responsive" with desired dimensions
4. Right-click > "Capture screenshot"

## Naming Convention

Use the exact filenames listed above for consistency with the documentation.
