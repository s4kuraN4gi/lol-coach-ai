# LoL Coach AI 🎮🤖

League of Legends の試合データを AI が自動分析してコーチングアドバイスを生成する Web アプリケーション。

---

## 🌟 概要

LoL Coach AI は Riot API を通じてプレイヤーの最新対戦データを取得し、  
OpenAI API を用いて「改善点」「練習タスク」「勝率アップのためのアドバイス」を生成します。

---

## 🧠 使用技術

- Next.js 14（App Router）
- TypeScript
- Tailwind CSS
- Riot API (summoner-v4 / match-v5)
- OpenAI API (gpt-4o-mini)
- pnpm

---

## 🧩 機能

- サモナーネーム入力による対戦データ取得（モックデータ対応）
- ロール選択によるロール別アドバイス生成
- AI出力を見やすくセクション分け表示
- 403/401エラーハンドリング

## LOLに必要な情報を分析
勝つためには？
・ネクサスを壊す
    タワーを破壊する
    インヒビの破壊
・タワーの破壊に必要
    ミニオンを押す
    ヘラルド・グラブを倒す

・障害
    敵チャンピオン
    敵ミニオン

・障害の撃破
    ミニオン→殴る
    敵チャンピオン→ファイトで勝つ

・敵チャンピオンに勝つには
    HP差、MP差を作る
        敵より多くスキル・AAを当てる・スキルを避ける
            敵と自分のチャンピオンの射程範囲を把握する
            ミニオンラストヒットのタイミングを狙う
        タワー下ファイトでタワーダメージを当てる
            相手にミニオンを押させて自陣タワー付近に持ち込む
        回復アイテムの使用
            ポーションの使用
    アイテム有利を作る
        敵よりお金を稼ぐ
        リコールタイミングを考える
        対面の相手に刺さるアイテムを購入する
    ミニオン差を作る
        プッシュウェーブを作る
            ラストヒットのみをとり、ミニオンの数を増やす
    数的有利を取る
        ミニマップの味方の位置でファイトか判断する
        ファイトの時に引く方向を事前に決めておく
        ピンを出して味方を寄せる
        TPを使用して味方に合わせる
    チャンピオン相性に勝つ
        使用しているチャンピオンが有利なファイトができる相手か考える
    ルーンを考える
        相手に刺さるルーンが何か考えて構成する
